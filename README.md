# BallDetectingRS

## Описание
___
Данный скрипт предназначен для трекинга положения мячика 
в настольном теннисе, а также фиксации момента падения мяча на стол.

___
## Реализация
___
- Создаются 2 потока, в каждом из которых происходит 
обработка видеопотока для обычной камеры и камеры глубины.
- Синхронизация потоков реализована через Singleton класс, 
оба потока передают и получают данные через единственный объект.

___
## ⚙️ Установка
___
### Требования
* Python 3.13+
* poetry

### Установка poetry
___
Windows 10/11 \

```bash
  (Invoke-WebRequest -Uri https://install.python-poetry.org -UseBasicParsing).Content | py -
```
При необходимости добавить переменные в path\
___
Linux
```bash
  curl -sSL https://install.python-poetry.org | python3 -
```
___
Проверка установки
```bash
  poetry --version
```

### Установка зависимостей
В дирректории с проектом
```bash
  poetry install
```
### Запуск проекта

```bash
  poetry run python ./main.py
```
